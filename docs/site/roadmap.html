Roadmap

Version 1 stays intentionally small.

The roadmap follows the specification strictly: finish the core item lifecycle and local notifications before anything else.

Implemented now

  • Dashboard
  • Add Item
  • Edit Item
  • Archive
  • Settings
  • Local notification scheduling
  • SwiftData persistence and EN/DE localization

Next up

  • Workflow-guided polish
  • Coverage hardening toward the 80% business-logic target
  • Build warning cleanup
  • Release preparation and App Store assets

Out of scope for v1

  • Barcode scanning
  • OCR
  • Widgets
  • Sync, analytics, sharing, recipes, AI features
